## 🏁 Getting Started

### Prerequisites

- Java JDK 8 or higher
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git for version control

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hasini-space/A24126510268-CP.git
cd A24126510268-CP
```

2. Compile Java solutions:
```bash
javac src/algorithms/**/*.java -d bin/
javac src/problems/**/*.java -d bin/
```

3. View the documentation:
```bash
# Open in your browser
open web/index.html  # macOS
start web/index.html # Windows
xdg-open web/index.html # Linux
```

## 💻 Usage

### Running a Solution

```bash
# Navigate to compiled binaries
cd bin/

# Run a specific problem solution
java problems.YourProblemClass
```

### Browsing Solutions

1. Open `web/index.html` in your browser
2. Navigate through the problem catalog
3. View detailed solutions with explanations
4. Check complexity analysis for each problem
